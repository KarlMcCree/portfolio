Fire Detection Robot

Overview

This project is an ESP32-based autonomous fire-detecting robot that can sense flames using IR flame sensors, detect gas leaks with an MQ-6 sensor, and extinguish fire using a relay-controlled water pump.

Technologies Used

  • ESP32
  • SIM800L GSM module
  • Flame Sensors, Gas Sensor
  • L298N Motor Driver
  • Firebase Integration

Features

  • Autonomous fire detection and extinguishing
  • Obstacle avoidance
  • SMS alerts via GSM module
  • Remote control via web interface
